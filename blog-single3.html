Tienda e-commerce Maliz StoreEN PROGRESO



Plantilla HTML de comercio electrónico de belleza es una plantilla de sitio web HTML altamente eficiente.

Esta plantilla de sitio web creativa y colorida ha creado una interfaz de élite junto con las mejores funcionalidades. Se puede vender productos para el cuidado del cabello, cuidado de la piel, productos para el cuidado facial: maquillaje, rímel, base, brillo de labios, sombra de ojos, delineador de ojos, bálsamo labial, imprimación+control de brillo, laca para el cabello+aerosol de acabado, fragancia, loción corporal y muchos más. Este sitio web receptivo está desarrollado con HTML5, CSS3, SCSS y Javascript.

Link de sitio web, prototipo:

https://maliz-store.github.io/tienda/


Actualmente este proyecto esta en desarrollo, se está trabajando con Visual Studio Code y haciento el front-end de la página web en un entorno localhost.
