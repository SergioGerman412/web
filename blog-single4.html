Portafolio Web: Sergio German QuispeEN PROGRESO



El presente portafolio web está desarrollado en HTML5, CSS3 y Javascript. Se compró un dominio propio en DonWeb y se ha usado un hosting gratuito 000webhost. También se ha subido el portafolio web a github mediante git, como un segundo hosting gratuito.

Una vez tenido la plantilla web, se levantó el portafolio web con Xampp en localhost:8080, y se modificó el index y afines con visual studio code.

Características:

- Esquema oscuro y claro

- HTML 5 y CSS 3

- Efecto de partículas

- Diseños adaptables

- Transiciones de supersmooth

- Pantalla y porcentaje de carga al abrir el sitio web
